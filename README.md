# 🐠 Notchquarium

**A living Frutiger Aero aquarium that hangs from your Mac's notch — and doubles as a system monitor.**

*Battery is the water level. CPU is the bubbles. RAM is the water clarity. Your busiest apps swim around as fish.*

</div>

---

> ⚠️ **Work in progress.** Built in the open. See [`docs/superpowers/specs`](docs/superpowers/specs) for the design and [`docs/superpowers/plans`](docs/superpowers/plans) for the build plan.

## Requirements

- macOS 14+ (a Mac with a notch is recommended)
- Swift 5.9+ toolchain

## Run

```bash
swift run Notchquarium
```
